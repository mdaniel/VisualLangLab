VisualLangLab - Download, Install, and Run

There are two files available for download. To get started quickly just download the all-inclusive jar file (immediately below here).

All-Inclusive JAR File (Quick-Start)

For all practical purposes, you only need VLL4J.jar to run VisualLangLab. Double-clicking this file starts the VisualLangLab GUI. The only pre-requisite is a 6.0+ JRE. Users with Unix-like systems will first need to perform a "chmod +x VLLS-All.jar".

You do not have to install any other software. This jar file also includes the Sample Grammars, but not the HTML documentation (which are in the VLL4J.zip file described below). It also includes the readme.txt, copying.txt, changes.txt, and licenses files described below.

  • readme.txt - Summary information about VisualLangLab
  • copying.txt - Copyright information
  • changes.txt - Changes from previous version
  • licenses - Directory containing copyright files

Full Binary-Release ZIP File (Planned, not Available)

The full binary release (VLL4J.zip) contains the following files/directories:

  • VLL4J.jar - the All-inclusive jar file described above
  • VLL4J-API.jar - a minimal library jar for supporting client programs
  • index.html - the entry-point for HTML documentation
  • docs - Directory containing documentation files
  • readme.txt - Summary information about VisualLangLab
  • copying.txt - Copyright information
  • changes.txt - Changes from previous version
  • licenses - Directory containing copyright files

There is no particular installation action required. After unzipping the fle to a directory, VisualLangLab can be started by just double-clicking VLL4J.jar as described for All-Inclusive JAR File above.
