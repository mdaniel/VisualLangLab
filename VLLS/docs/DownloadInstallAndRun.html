VisualLangLab - Download, Install, and Run

The latest binary release can be found in the Downloads area of the project page. Download the latest ZIP release, and extract the contents to a directory. You should have the following files and directories:

  • VLLS.jar - the run-time class-library
  • vlls.bat - a launcher for Windows. It assumes that VLLS.jar is also in the same directory
  • vlls - a launcher for UNIX-like environments. It assumes that VLLS.jar is also in the same directory
  • readme.txt - Summary information about VisualLangLab
  • copying.txt - Copyright information
  • changes.txt - Changes from previous version
  • index.html - Documentation in HTML
  • docs - Directory containing documentation files
  • licenses - Directory containing copyright files

There is no particular installation action required. After VLLS.jar is added to your classpath, VisualLangLab can be started by entering the following command: 

    scala vll.gui.VllGui

Alternatively the launcher scripts -- vlls.bat for Windows, and vlls for UNIX-like systems -- can also be used. Both launchers assume that VLLS.jar is also in the same directory.

Scala 2.9.0+ is a prerequisite for running this version. If required, you can download Scala from here.
