VisualLangLab - Download, Install, and Run

Information for the Java and Scala versions is provided separately below.

Java Version (10.xx)

Download

For all practical purposes, you only need to download the file VLL4J.jar to run VisualLangLab.

This jar file also includes the Sample Grammars, but not the HTML documentation. It also includes the readme.txt, copying.txt, changes.txt, and licenses files described below.

  • readme.txt - Summary information about VisualLangLab
  • copying.txt - Copyright information
  • changes.txt - Changes from previous version
  • licenses - Directory containing copyright files

Installation and Running

No specific installation action is required. Double-clicking VLL4J.jar starts the VisualLangLab GUI within which you can create, modify, run, and test grammars. Users on Unix-like systems may first need to perform a "chmod +x VLL4J.jar".

Alternatively, you may enter "java -jar VLL4J.jar" at a command prompt.

Prerequisites

The only pre-requisite is a 6.0 (or later) JRE.

Full Binary-Release ZIP File (Planned, not yet Available)

The full binary release (VLL4J.zip) contains the following files/directories:

  • VLL4J.jar - the All-inclusive jar file described above
  • VLL4J-API.jar - a minimal library jar for supporting client programs
  • index.html - the entry-point for HTML documentation
  • docs - Directory containing documentation files
  • readme.txt - Summary information about VisualLangLab
  • copying.txt - Copyright information
  • changes.txt - Changes from previous version
  • licenses - Directory containing copyright files

There is no particular installation action required. After unzipping the fle to a directory, VisualLangLab can be started by just double-clicking VLL4J.jar as described for All-Inclusive JAR File above.

Scala Version (7.xx)

The Java version, listed above, should be the preferred option for most users. It has all the features of the Scala version despite being a significantly smaller download.

The Scala version is only recommended for users with a non-negotiable need to write action-code in Scala. Version-7.xx files are available for download here.
