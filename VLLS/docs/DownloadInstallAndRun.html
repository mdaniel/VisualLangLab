VisualLangLab - Download, Install, and Run

The latest VisualLangLab binary release can be found in the Downloads area of the project page. Download the latest ZIP release, and extract the VLLS.jar file.

There is no particular installation action required. The VLLS.jar file must be on the classpath, and VisualLangLab can be started by entering the following command:

scala -classpath VLLS.jar vll.gui.VllGui

Scala 2.9.0+ is a prerequisite for running VisualLangLab. If required, you can download Scala from here.
