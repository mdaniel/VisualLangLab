VisualLangLab - Examples of Use

The following articles describe how VisualLangLab can be used to create parsers described elsewhere (and created by other authors using Scala parser-combinator code).

  • How do I feed input to a packrat parser? describes a parser that replicates the one discussed in this scala-user thread (as of 5th Nov 2011)
  • Rapid Prototyping for Scala Parser Combinators recreates the Payroll External DSL described on page 240 of Programming Scala
  • The Arith Expr Parser recreates the parser in 31.1 Example: Arithmetic expressions of Programming in Scala

For help with specific VisualLangLab topics or issues, use the FAQ.
