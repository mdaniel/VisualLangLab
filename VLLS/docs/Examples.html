VisualLangLab - Examples of Use

The following articles describe how VisualLangLab can be used to create parsers described elsewhere (and created by other authors using different tools or techniques).

  • A Java7 Grammar for VisualLangLab describes a Java7 grammar with actions that cause it to print out instances of the use of JDK7 project coin features
  • A Quick Tour describes some parsers inspired by examples in chapter 3, A Quick Tour for the Impatient, of the book The Definitive ANTLR Reference
  • Rapid Prototyping for Scala Parser Combinators recreates the Payroll External DSL described on page 240 of Programming Scala
  • The PS2E-Arith-Expr parser recreates the parser in chapter 31 of Programming in Scala
  • The PS2E-Simple-JSON parser recreates the JSON parser in chapter 31 of Programming in Scala
  • How do I feed input to a packrat parser? describes a parser that replicates the one discussed in this scala-user thread (as of 5th Nov 2011)

For help with specific VisualLangLab topics or issues, use the FAQ.
