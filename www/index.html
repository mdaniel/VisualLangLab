Project Description

VisualLangLab is an IDE for creating, testing, and running LL(*) parsers using Scala's parser combinators in a purely visual environment without code or grammar scripts of any kind. It is a rapid application development environment that enables you to quickly produce backtracking, recursive-descent LL(*) parsers without writing any code - and even without knowing how to program in Scala (well, almost). Parsers can be run and tested within the same environment.

Development with VisualLangLab is extremely fast as parser code-generation and compilation are not required. VisualLangLab uses Scala's parser combinator capabilities to build complex parsers on the fly at run-time, so an iterative, incremental design process works very well.

Downloading & running

You only need to download a single jar file: VisualLangLab.jar. You start the program by executing the following command:

scala -cp VisualLangLab.jar net.java.dev.vll.VisualLangLab

You must have Scala 2.8 (currently a Beta pre-release) installed. In turn, Scala 2.8 requires Java 1.5 or higher to be installed.
