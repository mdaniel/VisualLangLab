VisualLangLab Grammars

VisualLangLab stores grammar descriptions as XML files with with a "vll" extension. For example, the file containing the arithmetic expression parser is called "Arith.vll". A grammar files is opened in VisualLangLab by using the "File -> Open" menu, and is saved by using the "File -> Save" or "File -> SaveAs" menu items. The following grammars are currently available:
