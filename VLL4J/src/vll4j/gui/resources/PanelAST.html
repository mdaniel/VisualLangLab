The Abstract Syntax Tree (AST) is the data-structure produced by a parser.
A specific, predictable AST structure exists for any rule-tree, and is
defined using the following principles recursively to each node in the
rule-tree:
  • A Sequence yields an array of the ASTs of its child nodes.
  • A Choice yields a 2-element array. The first element is a 0-based
    index of the matching alternative (child node), and the second element
    is the AST returned by the matching branch
  • A RepSep yields a List containing the ASTs from repeated
    applications of the parser represented by the first child node.
    A RepSep with multiplicity * (0 or more) may return an empty List
